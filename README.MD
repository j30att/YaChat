# Yandex practicum 
## Chat app

__________
###Commands
`yarn dev` -- start dev server

`yarn buildClient` -- build frontend

`yarn buildServer` -- build node server

`yarn buildAll`: --build frontend and node server

`yarn start`: start node serve
